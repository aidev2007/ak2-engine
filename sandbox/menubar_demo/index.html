スクロール動作のデモ

このページを下にスクロールすると、ヘッダーが縮小し背景のブラーが強化されます。 スクロール量が 10px を超えると .is-scrolled クラスがヘッダーに付与され、 CSS トランジションによって高さ・シャドウ・背景色が滑らかに変化します。

transparentHeader: true を設定すると、ページ最上部では背景が完全に透明になります。 ヒーロー画像やフルスクリーン動画の上にヘッダーを重ねる場合に有効です。 このデモでは通常モードで表示しています。

ドロワーメニューを開いたままスクロールしようとすると、overflow: hidden によって ページスクロールがロックされます。スクロールバーの幅も自動的に補正されるため、 ドロワー展開時のレイアウトシフトは発生しません。

ナビゲーションのアクセシビリティにも配慮しています。トグルボタンには aria-label と aria-expanded 属性が実装されており、 ドロワーには aria-hidden が状態に応じて切り替わります。

グローバルナビの「underline」と「pill」バリアントは、ホバー追従インジケーターを 実装しています。マウスオーバーで滑らかに移動し、マウスアウト時には アクティブ項目の位置にリセットされます。
